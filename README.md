# srcshot

Windows 螢幕截圖工具，常駐系統匣，支援全域快捷鍵、截圖後即時標註與裁切。

## 功能

### 截圖模式

| 快捷鍵 | 模式 |
|--------|------|
| `Alt+Shift+R` | 拖曳框選區域 |
| `Alt+Shift+A` | 擷取目前作用中視窗 |
| `Alt+Shift+W` | 點選任意視窗 |

### 延遲擷取

系統匣右鍵選單可設定延遲（0 / 1 / 2 / 3 / 5 秒或自訂），方便擷取懸停選單、tooltip 等瞬間狀態。  
倒數期間螢幕中央顯示計時數字，並以橘色框標示即將擷取的區域。

### 編輯器

截圖後自動開啟編輯器：

| 工具 | 說明 |
|------|------|
| 筆 | 自由繪製 |
| 箭頭 | 繪製箭頭 |
| 矩形 | 繪製矩形框 |
| 文字 | 點擊後輸入文字 |
| 裁切 | 拖曳選取區域後立即裁切 |
| 複製 | 複製到剪貼簿 |
| 儲存 | 另存新檔對話框（記憶上次位置） |
| 復原 | 復原最後一筆畫或裁切 |

- `Esc` 關閉編輯器，不儲存
- 捲軸在圖片超出視窗時自動出現
- 滑鼠在畫布區自動顯示對應工具游標

### 設定（系統匣右鍵）

- **擷取滑鼠游標**：截圖是否包含游標
- **延遲擷取**：倒數秒數（支援自訂）
- 設定自動儲存至 `%APPDATA%\srcshot\settings.ini`
- 儲存路徑記憶至 `%APPDATA%\srcshot\last_dir.txt`

## 系統需求

- Windows 10 / 11（x64）
- Rust toolchain（用於從原始碼建置）

## 建置

```powershell
cargo build --release
```

產生的執行檔位於 `target\release\srcshot.exe`，無需安裝，直接執行即常駐系統匣。

## 技術

純 Rust + Win32 API（`windows` crate 0.58），不依賴 .NET 或第三方 UI 框架。
